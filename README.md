# DIF-AccumulationTool

> Unified tools landing: **https://enheragu.github.io/psychometric-tools/**

DIF-AccumulationTool is a static web application to simulate how small item-level DIF effects may accumulate in total scores for ordered polytomous tests.

It lets you:

- configure the number of items and response categories,
- define GRM-style `a` and `b` parameters per item,
- activate DIF for Group B,
- inspect category and cumulative curves,
- expand plots in a modal view,
- run a basic score accumulation simulation.
